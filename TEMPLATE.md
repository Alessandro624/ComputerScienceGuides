# Template per nuove guide

Usa questo template quando crei nuove guide nel repository.

- Titolo: breve e descrittivo
- Scopo: cosa copre la guida
- Prerequisiti: software/permessi richiesti
- Installazione/Setup: comandi per preparare l'ambiente
- Esempi/Comandi: esempi pratici e snippet utilizzabili
- Best practices: consigli e precauzioni
- Riferimenti: link esterni e documentazione ufficiale
- Autore / Data

## Esempio di struttura minimale

````md
# Titolo

## Scopo

Breve descrizione.

## Prerequisiti

- strumento X >= versione

## Setup

```bash
comando per installare
```

## Esempi

```bash
comando di esempio
```

## Riferimenti

- <https://esempio>
````
