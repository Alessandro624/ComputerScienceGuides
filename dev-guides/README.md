# Dev Guides

Questa cartella contiene guide pratiche sugli strumenti e le pratiche di sviluppo e amministrazione.

## Sotto-cartelle

- `testing/` — guide sui framework di test (es. JUnit)
- `databases/` — guide su database (MySQL, MariaDB)
- `debugging/` — strumenti di debugging (Valgrind, ecc.)

## Indice delle guide in questa cartella

- [Docker](docker.md): comandi e esempi per gestire immagini, container, volumi, reti e Compose.
- [Doxygen](doxygen.md): introduzione a Doxygen e generazione automatica della documentazione dal codice.
- [Git](git.md): elenco dei comandi Git più utili per workflow, branching e ripristino.
- [Linux](linux.md): comandi Linux utili, setup WSL e consigli per sviluppo su Linux/WSL.
- [Maven](maven.md): guida rapida a Maven, struttura del progetto e gestione dipendenze/plugin.
- [Mpi](mpi.md): introduzione a MPI per programmazione parallela e esempi di uso.
- [JUnit](testing/junit.md): guida rapida a JUnit 5 e best practice per i test.
- [MySQL - MariaDB](databases/mySQL_MariaDB.md): comandi essenziali, backup e gestione per MySQL/MariaDB.
- [Valgrind](debugging/valgrind.md): uso di Valgrind per trovare memory leak e profilare l'applicazione.
