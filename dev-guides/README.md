# Dev Guides

Questa cartella contiene guide pratiche sugli strumenti e le pratiche di sviluppo e amministrazione.

## Sotto-cartelle

- `build-tools/` - Strumenti di build (Maven, CMake, Makefile)
- `containerization/` - Containerizzazione (Docker, Kubernetes)
- `databases/` - Database (MySQL, MariaDB, PostgreSQL)
- `debugging/` - Strumenti di debugging (Valgrind)
- `documentation/` - Generazione documentazione (Doxygen)
- `os/` - Sistemi operativi e shell (Linux, Bash)
- `parallel-computing/` - Programmazione parallela (MPI)
- `testing/` - Framework di test (JUnit)
- `version-control/` - Controllo versione (Git)
